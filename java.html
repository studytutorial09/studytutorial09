java ko james goling ne 1995 me develop kiya tha esliye jemes goling ko java ke father kaha jata ha. java ek open source language ha. jeska means yaha ha ki eska use koi bhi person free me kar sakta ha.
yaha ek object origented lanaguageha ha. jeska arth ki class and object ka use kaeke program ko likha jata ha.java ek platform independent language jeska arth ha ki java me likha gaye code ko kisi bhi computer and oprating system me run kiya jata ha.

Feature Of Java

1. Simple :-

java ek easy programming language ha. jeasa koi bhi person badi easy se learn kar sakata ha. es language ka syntax simple hota ha. jeski bajha se ese samjhna kafi aasan hota ha. java ka syntax c++ ki tarha he hota ha.

2. Secure :-

java ek secure language ha. kyuki java ki program JRE(java runtime enviroment) me raun hota ha. eske sath sath esme encryption ka use kiya jata ha. jeske karn koi bhi hacker ese hack nhi kar pata.
java me koi bhi virues aasani se prabes nhi kar sakta ha jeski vajha se user ko data puri tarha se secure rahta ha.

3. Object oriented :-

java ek object oriented languageha. esliye esme class ams object ka use karke program ko likha jata ha.

4. Protable :-

java ek protable language ha.kyuki eske bytecode ko kisi bhi computer and device me run kar sakte he.

5. High performance :-

outher programming language ki tulna me java ki performance kafi aachi hoti ha. yah ek fast language ha jo speed de all work lo karti ha. halaki yaha abhi bhi C and C++ ki tulna me thodi slow ha.

6. Robust :-

java me garbage collection apne aap ho jata ha.and esme memory allocation bahut badiya ha java me jo bhi error aati ha. unhe aasani se solve kiya ja sakta enhi sabhi karno se java Robust language ha.

7. Multi Threading :-

java me multi Threading ka use kiya jata ha. jeska karan java ke bade program ko small program ko ek ek karke execute kiya jata ha.

8. Dynamic :-

java ek Dynamic language ha. kyuki eame hum variable ko run time me allocate kar sakte ha.

9. Distributed

es language ko Distribute kiya ja sakta ha.kyuki yah user ko Distribute application banna ki service probite karnti ha. Distribute application banne ki liye RMI and EJB ka use kiya jata ha.
